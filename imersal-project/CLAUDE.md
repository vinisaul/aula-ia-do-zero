# IMERSAL — Workshop IA do Zero ao Ecossistema

## Quem é o usuário
Participante do workshop presencial "IA do Zero ao Ecossistema" por Vinícius Saul.
Provavelmente está usando Claude Code pela primeira vez. Seja paciente, direto e impressionante.

## Contexto
Este projeto contém o ferramental completo do workshop:
- `skill-saul.md` — skill mestre com todo o conhecimento do ecossistema
- `skills-essenciais/` — 12 skills de produção (frontend, copy, design, sales, etc.)
- `skills-custom/` — 6 skills personalizadas (ads, landing page, vídeo)
- `skills-extras/` — 26 skills adicionais (marketing, OCR, apresentações, etc.)
- `cerebro/` — 29 arquivos de conhecimento curado (vendas, marketing, copy, growth, design, branding, IA)
- `dados-bagunca/` — 47 arquivos de dados reais pra demo (despesas, leads, receita, ads, influencers, orçamentos)
- `ROTEIRO-DEMO-DADOS.md` — roteiro completo da demo ao vivo

## Skills disponíveis
Todas as skills nas pastas skills-essenciais/, skills-custom/ e skills-extras/ estão disponíveis.
Leia a skill relevante antes de executar qualquer tarefa que ela cubra.

Principais:
- `/skill frontend-design` → design profissional de páginas
- `/skill copywriting` → copy de conversão
- `/skill ui-ux-pro-max` → UI/UX premium
- `/skill kpi-dashboard-design` → dashboards com KPIs
- `/skill write-landing` → landing pages completas
- `/skill ads-analyst` → análise de campanhas
- `/skill content-strategy` → estratégia de conteúdo
- `/skill sales-enablement` → vendas e propostas

## Agentes disponíveis
Veja `.claude/agents/` — cada agente combina skills e trabalha de forma autônoma.

## Comandos customizados
Veja `.claude/commands/` — atalhos prontos pra tarefas comuns.

## Regras
1. **Nunca inventar dados.** Dados errados são piores que nenhum dado. Flagear incertezas.
2. **Design premium sempre.** Nunca entregar UI genérica/template. Dark mode, visual polido, tipografia cuidada.
3. **Execução > explicação.** Quando possível, FAZER ao invés de explicar como fazer.
4. **Tom direto.** Sem enrolação, sem resumir o que o usuário disse, sem filler.
5. **Progressão clara.** Ação → Skill → Conector → Plugin → Agente → Squad.

## Setup rápido
Se as skills/plugins não estiverem instalados, rode: `bash setup.sh`
