## O que vamos fazer neste projeto

### 1. Briefing de reunião nível executivo
Quando eu pedir um briefing sobre alguém (pessoa, empresa), pesquisa TUDO: LinkedIn, site, Instagram, notícias, Google. Entrega um briefing nível consultor McKinsey:
- Quem é a pessoa (cargo, empresa, histórico)
- Empresa (tamanho, mercado, concorrentes, momento)
- Conexões em comum, interesses compartilhados
- Pontos de dor prováveis do negócio
- Ângulos de abordagem recomendados
- Perguntas inteligentes pra fazer na reunião
- O que NÃO falar / cuidados
- Resumo executivo de 3 linhas no topo

### 2. Lead generation com artefato visual
Pesquisa de leads reais com análise de gaps, score de oportunidade, mensagem personalizada. Artefato HTML interativo — não é planilha, é mini-sistema.

### 3. SaaS de gestão de serviços
Dashboard, CRM, pipeline, métricas de conversão, controle de custo, briefings, integração com email. Cada funcionalidade cresce em cima da anterior.

## Nível de entrega

CRÍTICO: Todo artefato gerado neste projeto deve sair com **nível de produção elite**. Não existe "depois a gente melhora". Já sai bom.

- Design premium. Referência: 21st.dev, Linear, Vercel Dashboard, Stripe, Notion.
- Typography com personalidade — Google Fonts distintas, não Arial/Inter/system
- Paleta de cores coerente com intenção (consulte os arquivos de design/paleta)
- Espaçamento generoso, breathing room
- Micro-interações, hover states, transições suaves
- Hierarquia visual clara — o olho sabe pra onde ir
- Métricas que ajudam a DECIDIR, não decoração
- Componentes com acabamento de produto real
- HTML/CSS/JS autossuficiente — abro no browser e funciona completo
- Ícones via CDN (Lucide, Heroicons, ou SVG inline)
- Gráficos via Chart.js ou similar quando aplicável
