## AARRR (Pirate Metrics)

```
A — Acquisition   → Como os usuários chegam?
A — Activation    → Tiveram boa primeira experiência?
R — Retention     → Voltam?
R — Revenue       → Pagam / pagam mais?
R — Referral      → Indicam outros?
```

**Erro clássico:** focar em acquisition quando o gargalo está em activation ou retention.
**Diagnóstico primeiro:** mapa o funil, mede cada etapa, ataca a maior queda.

### Por projeto

**New Bird ([[new-bird]])**
- Gargalo real: activation (lead agendado → comparece → fecha)
- Retention: LTV de clínica é alto se entregar resultado — foco em renovação de contrato
- Referral: indicação médico-para-médico tem coeficiente alto se explorado

**PocketSafe ([[pocket-safe]])**
- Gargalo atual: acquisition (ainda pré-lançamento)
- Referral: produto tem viralidade inerente (tag física é visível, gera curiosidade)
- Revenue: modelo one-time = coeficiente viral precisa compensar ausência de recorrência

## Growth Loops

Loop de crescimento é um ciclo onde output de uma etapa alimenta o input da próxima.

**Diferente de funil:** funil é linear (A→B→C→fim). Loop é circular (saída vira entrada).

### Tipos de loop

**Viral Loop (PocketSafe):**
> Usuário usa tag → amigo vê/pergunta → visita site → compra → usa tag → ...

**Conteúdo Loop (TikTok [[tiktok]]):**
> Posta conteúdo → ganha seguidores → mais alcance orgânico → mais views → credibilidade → converte em negócio → financia mais conteúdo

**Paid Loop (New Bird):**
> Investe em ad → gera lead → fecha cliente → cliente paga → reinveste em ad → ...

Loop pago só funciona se LTV/CAC ≥ 3x. Abaixo disso, o loop drena caixa.

---

## Coeficiente Viral (K)

```
K = convites por usuário × taxa de conversão dos convites
K > 1 → crescimento exponencial
K < 1 → crescimento linear (precisa de acquisition externa)
```

**Para PocketSafe:** estimativa K = 1,3–1,8 se produto for bem distribuído (tag visível = convite passivo).

## North Star Metric

Uma métrica que resume melhor que qualquer outra se o produto/negócio está criando valor.

| Negócio | North Star |
|---|---|
| New Bird | Agendamentos gerados por mês |
| PocketSafe | Tags ativas (ativadas pelo usuário) |
| Carajillo | Pontos de venda ativos |
| TikTok pessoal | Seguidores qualificados acumulados |

**Regra:** se a North Star sobe, a empresa cresce. Se não tem North Star definida, é difícil priorizar.

## Retention — por que é o mais importante

**LTV = Ticket × Retention**

Dobrar retention tem mais impacto que dobrar acquisition na maioria dos negócios maduros.

**Churn composto:** 10% churn mensal = 72% dos clientes perdidos em 12 meses.
**Retention composto:** 90% de retenção mensal = 28% perdidos. Diferença radical.

**Táticas de retention para agência (New Bird):**
- Relatório mensal com número claro de resultado (não atividade)
- QBR (Quarterly Business Review) com cliente — conecta estratégia
- Expansão de serviço (upsell) antes do cliente cogitar sair

## Experimentos de Growth

**Framework ICE (priorização):**
- **I**mpact: quanto move a North Star se funcionar?
- **C**onfidence: qual a probabilidade de funcionar?
- **E**ase: qual o esforço de implementar?

Score = (I × C × E) / 3. Roda experimentos em ordem decrescente.

**Teste mínimo válido:** hipótese clara, métrica de sucesso definida antes, prazo fixo.
Sem hipótese antes, é tweaking aleatório, não growth.
